# HotelSQLite

Sistema sencillo de gestión de hoteles y facturación.
Creado para la asignatura de Desarrollo de Interfaces

## Instalar

Instalar dependencias con `pip3 -r requirements.txt`.

## Ejecutar

Ejecutar con `python3 main.py`.

## Tecnologías

Python 3
Glade (Gtk-3)
